# VISION: Fallout RPG Web Companion

Una web 100% frontend (vanilla HTML+CSS+JS) que sirve como acompañante de mesa para el juego de rol Fallout 2d20 de Modiphius. La web permitirá a los jugadores consultar rápidamente reglas, skills, perks y equipo con todas sus opciones (armas, calibres, mods, armaduras, consumibles), además de incluir un creador de personajes básico. Todo funcionará con datos en JSON, sin backend, desplegada en GitHub Pages. Más adelante se añadirán herramientas como generador de ciudades, NPCs y encuentros.
