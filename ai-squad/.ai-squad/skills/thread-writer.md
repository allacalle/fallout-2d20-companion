# Thread Writer

**Trigger**: "create a thread", "write a thread", "Twitter thread", "X thread", "thread about"

**Dependencies**: Reads `about-brand.md` and `brand-voice.md` if they exist

## Description

Writes Twitter/X threads of 3-7 tweets in your brand voice.

## Step 1: Gather Inputs

1. **What topic?** (or paste notes/article)
2. **Length?** 3, 5 or 7 tweets
3. **Goal?** Education, storytelling, promotion, controversy

## Step 2: Thread Structure

**Tweet 1 - Hook**:
- Max 280 chars
- Hook that forces readers to continue
- Don't reveal all value upfront

**Tweets 2 to N-1 - Body**:
- 1 idea per tweet
- Max 280 chars each
- Natural transitions between tweets
- Use numbers, lists, examples

**Tweet N - Close**:
- Summary or takeaway
- CTA if applicable (follow, comment, link)
- Don't repeat the hook

## Step 3: Write

Rules:
- Each tweet must work standalone BUT invite to the next
- Use line breaks for readability
- Emojis only if brand-voice.md allows
- Numbers > spelled words

Output in code block:

```
Tweet 1/5:
[content]

Tweet 2/5:
[content]

...
```

## Step 4: Iterate

> How does this feel? Tell me what to change or say "ready" and I'll save.

## Rules
- Max 280 chars per tweet. COUNT.
- No numbering inside the tweet (only "1/N" at start)
- English unless specified otherwise
- Save to `.empresa/content/threads/[date]-[topic].md`
