Compendio web de consulta rápida para el juego de mesa **Fallout 2d20** (publicado por Modiphius).

## Funcionalidades

- Compendio de equipo: armas, armaduras, ropa, atuendos, headgear, dog armor, consumibles, munición, libros y revistas, objetos varios
- Modificaciones de armas y armaduras
- Armaduras de robot y sus mods
- Crafteo: recetas de química, cocina y armadura de poder
- Habilidades y perks
- Glosario de efectos, cualidades y localizaciones

## Cómo usar

- **Online**: https://allacalle.github.io/fallout-2d20-companion/
- **Local**: Necesitas un servidor HTTP (el navegador bloquea `fetch()` con `file://`):

  ```bash
  python3 -m http.server 8000
  ```

  Luego abre `http://localhost:8000` en el navegador.

## Tecnología

HTML, CSS y JavaScript vanilla. Sin dependencias externas. Sin backend. Sin recolección de datos.
